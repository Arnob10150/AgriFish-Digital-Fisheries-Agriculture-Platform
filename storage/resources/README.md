# Fish Images for Marketplace

This folder contains images for the fish products displayed in the marketplace.

## Image Path
Images are accessed via: `http://localhost/DFAP/storage/resources/images/fish/`

## Expected Image Files

Place the following image files in `DFAP/storage/resources/images/fish/`:

### Freshwater Fish:
- `ilish.jpg` - Ilish (Hilsa)
- `rui.jpg` - Rui (River)
- `katla.jpg` - Katla (River)
- `ayre.jpg` - Ayre (Giant Catfish)
- `chitol.jpg` - Chitol (Featherback)
- `boal.jpg` - Boal (Wallago)
- `shing.jpg` - Shing (Stinging Catfish)
- `pabda.jpg` - Pabda (Pabo Catfish)

### Sea Fish:
- `rupchanda.jpg` - Rupchanda (Pomfret)
- `koral.jpg` - Koral (Seabass)
- `tuna.jpg` - Tuna
- `loitta.jpg` - Loitta (Bombay Duck)
- `surma.jpg` - Surma (King Fish)
- `poa.jpg` - Poa (Yellow Croaker)

### Shellfish:
- `golda-chingri.jpg` - Golda Chingri (Prawn)
- `bagda-shrimp.jpg` - Bagda/Tiger Shrimp
- `lobster.jpg` - Lobster
- `crab.jpg` - Crab (Mud/Blue)

### Other:
- `churi-shutki.jpg` - Churi Shutki (Dried)
- `basa-fillet.jpg` - Basa/Dory Fillet

## Image Specifications:
- **Format**: JPG, PNG
- **Size**: 300x300px recommended
- **Quality**: High quality product photos
- **Naming**: Use lowercase with hyphens (no spaces)

## Fallback:
If images are not found, the system will display emoji placeholders.
